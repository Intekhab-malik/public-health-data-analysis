# Oral Health Data Analysis (R)

## Project Aim
To explore oral health inequalities in England using publicly available
oral health and deprivation data.

## Background
Oral health inequalities remain a major public health concern in England.
This project applies basic epidemiological and data analysis methods
to understand variation in oral health outcomes.

## Data Source
- Public Health England / OHID oral health indicators
- Index of Multiple Deprivation (IMD)

## Methods
- Data cleaning in R
- Descriptive statistics
- Simple visualisations

## Tools
- R
- GitHub
